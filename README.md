# AstrBot Server Ops Agent (v1.2.0)

基于 LLM 的远程服务器运维助手，通过自然语言对话即可实现复杂的 Linux 服务器管理。

## 🌟 核心特性

- **自然语言运维**：直接告诉 Bot "帮我看看系统负载"、"帮我安装 nginx" 或 "重启 docker 容器"。
- **可视化输出**：自主识别输出长度和类型，将目录树 (`ls -R`)、长日志 (`tail`) 等渲染为精美图片发送。
- **引用多媒体支持**：可以将用户在聊天中发送的图片、固件、脚本等文件，直接按需下载并传输至服务器指定位置。
- **智能 Skill 记忆**：Agent 会将复杂或常用的高级操作流程（如 blog 发布 API 流程）记入长期记忆，排除基础探测命令。
- **对话隔离系统**：运维对话记录与主聊天窗口完全隔离，支持独立清除和查看。
- **零依赖配置**：无需在服务器端安装 Agent，仅需 SSH 访问权限，支持密码及私钥。

## 📸 功能演示

### 1. 媒体文件传输
引用一张图片并说：`/ops 把这张图片下载到服务器 /var/www/html/dist/bg.jpg`，Agent 会自动提取 URL 并在服务器端搞定。

### 2. 精准技能记忆
发送 `/ops 记住如何通过 API 发布 hexo 博客`，Bot 会记录下完整的 API 调用细节，而不是它刚才为了找路径执行的 `ls` 命令。

## ⌨️ 常用命令

- `/ops <任务>`：启动运维 Agent 执行任务。
- `/ops_test`：启动 SSH 连接诊断。
- `/ops_skills`：查看已学到的技能库。
- `/ops_forget <名称>`：遗忘特定技能。
- `/ops_log`：查看隔离的运维历史记录。
